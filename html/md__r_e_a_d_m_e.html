OxShmolli2 and Numerical Recipes

Important Original ShMOLLI (and based on it OxShmolli2) uses code based on Numerical Recipes book. Due to Numerical Recipes licence I cannot share this part of the code online. Please contact me if you would like to use this part of the code. To make up for this limitation we provide a number of alternative fitting procedures.

Coding style

  • Underscore (_variable) means a class member variable.
  • Define copy constructors. In copy constructor copy primitive types, set all pointers to zero.
